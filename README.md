# pronosticos-app
Aplicación para obtener pronósticos de ventas en ELTEC

- Python 3.9.1 (descargar desde https://www.python.org/downloads/)
- python Pandas
- python lxml
  
Para instalar:
```
pip install pandas
pip install lxml
```
Por defecto en python:
- python os
- python http.server

Para correr el server:

```{python}
py server.py
```

Corre en el puerto 8091
